# Cubic B√©zier

An easing curve as defined in [8.6](https://design-tokens.github.io/community-group/format/#cubic-bezier).

::: code-group

```json [JSON]
{
  "linear": {
    "$type": "cubicBezier",
    "$value": [0, 0, 1, 1]
  },
  "easeOutCubic": {
    "$type": "cubicBezier",
    "$value": [0.33, 1, 0.68, 1]
  },
  "easeInCubic": {
    "$type": "cubicBezier",
    "$value": [0.32, 0, 0.67, 0]
  },
  "easeInOutCubic": {
    "$type": "cubicBezier",
    "$value": [0.65, 0, 0.35, 1]
  }
}
```

```yaml [YAML]
linear:
  $type: cubicBezier
  $value:
    - 0
    - 0
    - 1
    - 1
easeOutCubic:
  $type: cubicBezier
  $value:
    - 0.33
    - 1
    - 0.68
    - 1
easeInCubic:
  $type: cubicBezier
  $value:
    - 0.32
    - 0
    - 0.67
    - 0
easeInOutCubic:
  $type: cubicBezier
  $value:
    - 0.65
    - 0
    - 0.35
    - 1
```
